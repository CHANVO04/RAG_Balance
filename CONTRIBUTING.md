# Hướng dẫn đóng góp cho dự án (Contributing)

Chào bạn! Rất vui vì bạn muốn đóng góp để dự án này tốt hơn. Dù bạn là lập trình viên chuyên nghiệp hay mới học code, mọi đóng góp của bạn đều rất đáng quý.

## Các cách bạn có thể đóng góp:
1. **Báo lỗi**: Nếu bạn chạy dự án mà gặp lỗi hoặc thấy chỗ nào chạy không đúng, hãy mở một "Issue" để báo cho mình biết.
2. **Gợi ý tính năng mới**: Nếu bạn muốn dự án có thêm tính năng gì hay ho, bạn cũng có thể mở "Issue" để thảo luận.
3. **Sửa code**: Nếu bạn biết cách sửa lỗi hoặc tự code được tính năng mới, bạn hãy fork dự án về, sửa code và gửi "Pull Request (PR)".

## Quy trình gửi Pull Request (PR):
1. Fork dự án này về tài khoản GitHub của bạn.
2. Clone dự án từ tài khoản của bạn xuống máy tính cá nhân.
3. Tạo một nhánh (branch) mới để sửa code (ví dụ: `git checkout -b sua-loi-gi-do`).
4. Sửa code và chạy thử trên máy của bạn xem có chạy tốt không.
5. Commit và push code lên nhánh mới của bạn trên GitHub.
6. Quay lại repo gốc này và nhấn nút "New Pull Request" để gửi code cho mình duyệt.

Cảm ơn bạn rất nhiều!
